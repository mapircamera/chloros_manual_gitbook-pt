# Introdução

<div data-full-width="false"><figure><img src=".gitbook/assets/chloros_logo_transparent.png" alt=""><figcaption></figcaption></figure></div>

O Chloros é um aplicativo de software da [MAPIR](https://www.mapir.camera) para processar imagens e outros dados de sensores.

***

O Chloros está disponível em 4 modos de aplicação:

## Chloros: Aplicação GUI para ambiente de trabalho

\[Recomendado] Janela separada independente com todas as funcionalidades.

## Chloros (Navegador): Interface GUI baseada na Web

Executa-se no seu navegador da Web (Chrome/Edge). Ideal para computadores mais lentos com &lt;4 GB de VRAM.

## [Chloros CLI: Interface de linha de comando](CLI.md)

Processamento em lote por linha de comando. Perfeito para automação, scripts e fluxos de trabalho avançados. _A CLI requer uma licença Chloros+ para acesso._

## [Chloros API: Python SDK](api-python-sdk.md)

Interface programática Python para automação e fluxos de trabalho personalizados. Perfeita para pipelines de pesquisa, integração com aplicações Python existentes e criação de ferramentas personalizadas. _A API requer uma licença Chloros+ para acesso._

***

## Chloros+

Embora o Chloros seja gratuito para a maioria das tarefas, poderá querer mais. É aí que uma licença paga para o Chloros+ pode ser vantajosa. Com uma licença Chloros+, pode desbloquear novas funcionalidades, tais como:

* **Processamento multithread**: acelere significativamente o processamento de imagens para projetos maiores, processando imagens simultaneamente através do pipeline.
* **Aceleração GPU (CUDA)**: aproveite as opções de memória GPU mais altas atuais para acelerar ainda mais o pipeline de processamento de imagens. Recomendamos 4 GB ou mais de VRAM para obter melhores resultados.
* **Chloros+** [**CLI**](CLI.md) **Acesso**: execute o Chloros+ a partir da linha de comando para automatizar e integrar no seu próprio software.
* **Chloros+** [**API**](api-python-sdk.md) **Acesso:** execute o Chloros+ a partir do Python para controlo programático, permitindo uma integração perfeita com os seus pipelines de pesquisa, fluxos de trabalho de análise de dados e aplicações personalizadas.
* **Utilização em vários dispositivos**: cada licença do Chloros+ permite o registo de mais de dois dispositivos. Utilize a sua conta MAPIR Cloud para gerir os dispositivos registados. Adicione suporte para mais dispositivos atualizando a sua licença do Chloros+.
* **Fórmulas personalizadas de índices multiespectrais:** introduza índices multiespectrais personalizados nas calculadoras raster Chloros, tanto para processamento como para a área de visualização de imagens.
